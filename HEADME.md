# RecyclerView - Message List (Kotlin)

Projeto Android simples demonstrando o uso de RecyclerView com Adapter e ViewHolder.

## Tecnologias
- Kotlin
- RecyclerView
- Android SDK

## Funcionalidades
- Listagem de mensagens
- Adapter customizado
- ViewHolder para otimização de performance

## Estrutura do Projeto
- Mensagem.kt: modelo de dados da mensagem
- MensagemAdapter.kt: Adapter do RecyclerView
- RecyclerViewActivity.kt: configuração e exibição da lista
- item_lista.xml: layout do item da lista

## Objetivo
Projeto educacional focado em consolidar os fundamentos de RecyclerView no Android.
